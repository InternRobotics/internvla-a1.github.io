Unified Understanding-Generation-Action Framework

The framework consists of:
(1) an understanding expert that parses image and text inputs to encode scene context;
(2) a generation expert that predicts future visual states and task dynamics;
(3) an action expert that utilizes these predictions to generate control commands via Flow Matching.
This tripartite architecture enables robust and flexible autonomous manipulation.
